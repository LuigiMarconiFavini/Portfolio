# 💼 Portfolio - Luigi Marconi Favini

Portfolio profesional desarrollado para presentar mis proyectos, tecnologías y perfil como desarrollador Full Stack.

## 🚀 Sobre el proyecto

Aplicación web desarrollada con React + Vite, enfocada en:

- Diseño moderno y responsive
- 
- Sistema de idioma dinámico (Español / Inglés)
- 
- Sección de proyectos con enlaces a repositorios
- 
- Descarga directa de CV
- 
- Animaciones y experiencia de usuario cuidada

## 🛠 Tecnologías utilizadas

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3
- Framer Motion
